# Blog V2 重构执行说明（2026-03-09）

## 本轮目标

严格按高保真 V2 设计稿，把博客从旧的深色科技感/组件站气质，整体切换为：

- 暖米色纸感背景
- 深棕文字与低饱和强调色
- 半透明玻璃卡片
- 克制、安静、专业的中文个人博客风格
- 文章优先、阅读优先，而不是功能优先

## 设计决策摘要

### 1. 不做“首页像稿子，内页像旧站”
所以这轮重点是把列表页、详情页、归档页、随记页全部接入同一套语言，而不是只修首页。

### 2. 文章与时间信息更重要
文章卡片、详情页 meta、归档结构都强化了“这篇写什么 / 什么时候写 / 要读多久”的信息，而不是做功能展示。

### 3. 纸感 + 玻璃卡片并存
保留高保真稿里的轻玻璃感，但整体基调改为更适合承载中文长文阅读的暖纸面体系。

### 4. 详情页优先服务阅读
文章详情页和随记详情页都控制了阅读宽度、正文节奏、目录层级和继续阅读模块，尽量靠近成熟个人博客的阅读体验。
